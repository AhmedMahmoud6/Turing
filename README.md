# TEDxITECH - Greek Campus MOA Event

This project is part of the **TEDxITECH** student activity, focused on organizing and managing an upcoming event that will take place at the **Greek Campus MOA**. The web application is designed to provide event details, registration, and other essential features to enhance the attendee experience.
